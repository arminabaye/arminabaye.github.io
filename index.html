resist.network

A website to find/post peaceful protests in your community.


About the Project:

I created this website to support the racial justice movement of 2020. Information regarding events was spread out across different social media platforms, making it hard to keep track. The solution: a single platform to create and view events.

Technical Features:

  • Created in WordPress with modified CSS
  • Event creation page in JavaScript
  • Map View uses Google Maps API
  • 65 events and 5000 unique users in first 14 days
