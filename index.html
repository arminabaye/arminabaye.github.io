resist.network

A website to find/post peaceful protests in your community.


About the Project:

I created this website to support the racial justice movement of 2020. Information regarding events was spread out across different social media platforms, making it hard to keep track. The solution: a single platform to create and view events.

Technical Features:

  • Created in WordPress with modified CSS
  • Event creation page in JavaScript
  • Map View uses Google Maps API
  • 65 events and 5000 unique users in first 14 days

Zot Spot

A mobile application to find and reserve parking at UCI.


About the Project:

  • Created as part of a team for a quarter-long project course in INF 132 following the entire UX process.
  • Click to view an interactive prototype of ZotSpot!



Gathering Data:

The first step in the design process was creating user research surveys for a competitive analysis. We targeted our main demographic group (UCI Students) and surveyed them about their experience with UCI parking.

We found that there was no existing application to reserve parking spaces at UCI and students were struggling with the on-campus parking experience.
